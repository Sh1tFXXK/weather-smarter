# 后端启动与自检说明

## 常驻启动

在项目根目录执行：

```cmd
scripts\run_server_local.cmd
```

脚本会自动：

- 注入本地 `llama_cpp` 模型配置
- 注入真实天气上游 `UAPIS` 配置
- 后台启动 `uvicorn`
- 打印前端访问地址 `http://127.0.0.1:8000/`
- 自动执行 HTTP 自检

## 单独检查

```cmd
.\.pyembed\python.exe scripts\check_backend_http.py
```

检查项包括：

- `/docs`
- `/api/v1/status`
- `/api/v1/tasks`
- `/api/v1/query`
- `/api/v1/misc/weather`
- `/api/v1/forecast`
- `/api/v1/aqi`

默认会校验：

- `/query` 已进入 LLM 增强模式
- 天气相关接口不再返回 `mock`

## 前端打开方式

优先使用：

```text
http://127.0.0.1:8000/
```

不要优先直接双击打开 `frontend/index.html`。如果确实直接打开了本地 HTML，页面现在也会自动改连到 `http://127.0.0.1:8000`，并在顶部显示连接状态。

## 单独探针

验证本地模型：

```cmd
.\.pyembed\python.exe scripts\probe_llm_runtime.py
```

验证天气上游：

```cmd
.\.pyembed\python.exe scripts\probe_weather_upstream.py
```

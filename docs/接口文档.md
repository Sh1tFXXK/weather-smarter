# Weather Smarter 接口文档（v1）

面向生活决策的自然语言交互式天气服务系统后端 API。默认返回 JSON（UTF‑8），时间采用 ISO‑8601。

## 基础信息
- Base URL（本地开发）：`http://localhost:8000`
- 版本：所有路由均以 `/api/v1` 开头
- 认证：`Authorization: Bearer <token>` 或 `X-API-Key: <key>`（本地可关闭）
- Content-Type：请求 `application/json`；SSE `text/event-stream`
- 频率限制：可能返回 `X-RateLimit-*` 与 `Retry-After` 头

## 通用约定
- 地理：`lat`、`lon` 为小数度；也可传 `place` 名称（需地理编码）
- 单位：`units=metric|imperial`（默认 metric）
- 语言：`lang`（默认 `zh-CN`）
- 时区：`tz`（IANA，如 `Asia/Shanghai`），未传则按位置推断
- 错误响应：`{ "error": { "code": "...", "message": "...", "details": {...} }, "traceId": "..." }`

## 1) 自然语言决策
POST `/api/v1/query`
- 用途：将自然语言问题解析为意图与实体，结合天气/规则/记忆生成可解释建议
- 请求体：
```json
{
  "query": "明早7点从浦东骑行通勤，需要带雨衣吗？",
  "locale": "zh-CN",
  "sessionId": "abc-123",
  "location": {"lat": 31.23, "lon": 121.53},
  "time": "2026-03-13T07:00:00+08:00",
  "scenario": "commute",
  "units": "metric"
}
```
- 成功响应示例：
```json
{
  "intent": "commute",
  "entities": {
    "time": "2026-03-13T07:00:00+08:00",
    "mode": "bike",
    "location": {"name": "上海·浦东", "lat": 31.23, "lon": 121.53}
  },
  "forecastSummary": "小雨，体感12℃，北风3级，能见度良好",
  "decision": {
    "advice": "建议携带轻便雨衣并加装挡泥板。",
    "riskLevel": "medium",
    "reasons": ["降水概率56%","路面湿滑"],
    "actions": ["随身雨衣","提前10分钟出发"]
  },
  "followUps": [],
  "sources": ["weather-provider:forecast/v1"],
  "traceId": "req_123"
}
```
- 流式（SSE，可选）：GET `/api/v1/query/stream?query=...&locale=zh-CN&lat=...&lon=...`，事件：`chunk`、`final`、`error`

## 2) 天气预报（直通）
GET `/api/v1/forecast`
- 参数：`lat`（必传）、`lon`（必传）、`hours`（默认 24）、`days`（默认 7）、`units`、`lang`、`tz`
- 响应示例（节选）：
```json
{
  "location": {"lat": 31.23, "lon": 121.53, "tz": "Asia/Shanghai"},
  "current": {"ts": "2026-03-12T10:00:00+08:00", "temp": 14.2, "wind": 3, "aqi": 55},
  "hourly": [{"ts": "2026-03-12T11:00:00+08:00", "temp": 14.5, "pop": 0.4}],
  "daily": [{"date": "2026-03-13", "tmin": 11, "tmax": 16, "uv": 4}]
}
```

## 3) 空气质量
GET `/api/v1/aqi?lat=..&lon=..&units=metric`
- 返回最近时刻 AQI 与主要污染物（与天气同源缓存）

## 4) 任务/提醒
POST `/api/v1/tasks`
- 用途：创建与天气相关的提醒（例如“明早跑步看天气”）
- 头：建议传 `Idempotency-Key: <uuid>`
- 请求体：
```json
{
  "type": "sport",
  "scheduled_time": "2026-03-13T06:30:00+08:00",
  "priority": 5,
  "metadata": {"location": {"lat": 31.23, "lon": 121.53}}
}
```
- 响应：`{ "task_id": 1024, "status": "scheduled" }`

## 5) 健康检查
- GET `/health` → `{ "status": "ok" }`
- GET `/metrics` → Prometheus 指标（如启用）

## 错误码（示例）
- `VALIDATION_ERROR`（400）：参数不合法/缺失
- `UNAUTHORIZED`（401）：认证失败/缺少密钥
- `NOT_FOUND`（404）：资源不存在
- `RATE_LIMITED`（429）：频率受限（含 `Retry-After`）
- `UPSTREAM_ERROR`（502）：上游天气/地理服务错误
- `SERVER_ERROR`（500）：未知错误

## cURL 示例
- 决策：
```bash
curl -X POST http://localhost:8000/api/v1/query \
  -H "Content-Type: application/json" \
  -d '{
    "query":"明早7点从浦东骑行通勤，需要带雨衣吗？",
    "locale":"zh-CN",
    "location":{"lat":31.23,"lon":121.53},
    "units":"metric"
  }'
```
- 预报：
```bash
curl "http://localhost:8000/api/v1/forecast?lat=31.23&lon=121.53&hours=24&units=metric"
```

> 说明：字段与枚举以实际实现为准（Pydantic 校验）。返回包含 `traceId` 便于排障；服务可能追加 `X-API-Version` 与弃用头以提示变更。
